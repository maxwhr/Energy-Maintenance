## 规则

抽取器同时识别有上下文约束的数字码、字母数字码、连字符代码、告警名称、指示灯状态与故障现象标题。官方内容只有名称而没有代码时写入 `alarm_identifier_type=name_only`；不会根据序号、页码或型号猜测告警码。每条知识保留适用型号和 source locator。

## 已知缺口

当前统计是 pending 候选而非 active formal 知识。SmartGuard 专项告警文档仍缺失；5 份短 FAQ 因元数据/最小正文长度不足未进入可批量审核集合。正式覆盖只能在人工批准后由 corpus gate 重算。
